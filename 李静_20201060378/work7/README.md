# GLUT鼠标函数和反走样技术实验报告

## 学号：20201060378

## 实验目的
1. 熟悉鼠标函数
2. 熟悉反走样技术的概念

## 实验内容
1. 参考课本实现鼠标函数（2个）
2. 使用opengl，实现任一反走样技术

## 实现功能
work7.cpp实现了在窗口每次按下鼠标左键，就在鼠标光标所在位置画一个尺寸为3的红点的功能
work7_1.cpp实现了用鼠标输入来选择直线段端点位置，选中的直线段首尾相连，交互生成折线的功能。在初始时，必须用鼠标左键在窗口内选择两个位置来生成第一个直线段。
work7_2.cpp实现了将走样图形反走样的功能
