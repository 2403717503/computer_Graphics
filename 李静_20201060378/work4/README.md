# 扫描线填充算法填充多边形实验报告

## 学号：20201060378

## 实验目的
1. 熟悉多边形的填充
2. 验证扫描线填充算法
3. 掌握扫描线填充算法的原理

## 实验内容
1. 参考课本代码，完成六边形的填充
2. 参考课本代码，实现扫描线算法填充任意多边形

## 实现功能
work4.cpp实现将给定六边形填充的功能
work4.1.cpp实现输入四个顶点，将这四个点依次连接，形成多边形，并将其填充成绿色的功能

## 输入数据
work4.cpp无输入数据
work4.1.cpp输入多边形顶点：(250,50),(550,150）,（550,400）,（250,250）
