# 三维观察实验报告

## 实验目的
1. 熟悉三维观察的流程

## 实验内容
1. 参考课本代码实现正方形的透视投影视图

## 实现功能
work12.cpp实现了将在xy平面的正方形，通过确定其中心为注视点，得到它的一个透视视图
work12_1.cpp实现了通过按键A，S实现三棱锥的顺时针，逆时针的旋转操作，展现了三棱锥的各个角度的外观
